# Лабораторная работа №2 | Вариант 33
## Визуализация и спектральный анализ речевых сигналов

### Описание
Программа выполняет анализ речевых сигналов из WAV-файлов и строит четыре типа визуализации:

1. **Линейный график с маркерами '*'** — временное представление сигнала с выделением каждого отсчёта
2. **Осциллограмма** — классическая форма сигнала по времени
3. **Спектр: мнимая часть ДПФ (Jm)** — фазовые характеристики частот
4. **Гистограмма** — статистическое распределение амплитуд

### Требования к WAV-файлу
- Формат: PCM signed 16-bit
- Каналы: моно (или стерео — будет использован левый канал)
- Частота дискретизации: 48000 Гц (для варианта 33)
- Длительность: не более 3 секунд
- Содержание: запись собственной речи с именем и фамилией

### Установка зависимостей
```bash
pip install numpy matplotlib
```

### Использование
1. Запустите программу:
```bash
python 2lab.py
```

2. Введите имя WAV-файла (например: `33.wav`)

3. Выберите количество отсчётов для отображения на всех графиках

4. Просмотрите построенные графики

5. Программа выведет время выполнения вычислений

### Особенности варианта 33
- **График 1**: Линейный сплошной с маркерами '*' на каждой точке
- **График 3**: Мнимая часть ДПФ вместо модуля или логарифма
- **Частота**: 48000 Гц для качественного анализа речи
- **Единая логика**: все графики строятся по одинаковому количеству отсчётов

### Особенности программы
- **Обработка ошибок**: программа не падает при некорректном вводе
- **Поддержка разных форматов**: 8-bit, 16-bit, 32-bit WAV-файлы
- **Graceful exit**: корректная обработка Ctrl+C и Ctrl+Z
- **Универсальность**: работает с моно и стерео файлами
- **Измерение производительности**: время выполнения вычислений

### Структура кода
- Чтение и валидация WAV-файла
- Преобразование аудиоданных в числовой массив  
- Единообразная обработка всех графиков
- Спектральный анализ через дискретное преобразование Фурье
- Измерение времени только вычислительных операций

### Результат выполнения
Программа выводит:
- Информацию о WAV-файле (частота, разрядность, длительность)
- 4 графика согласно варианту 33
- Время выполнения вычислений в секундах
